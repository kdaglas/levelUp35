# levelUp35

### Assignment one

- Create a new repository with a new name.
- The repository should have a readMe file.
- The repository should have two branches that is `master` and `develop`.
- Link the Github repository to your pivotal tracker board and make sure all Github activity is reflected in the relevant pivotal tracker story.
- Add `index.html` and `style.css` files and push them to Github with descriptive commit messages.
- You should create a `pull request` from your `develop` branch to your `master`` branch
- Request at-least two of your friends to review your pull request
- Merge your `pull request` to master.
